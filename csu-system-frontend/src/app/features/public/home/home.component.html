¿Qué es ASIS?

El sistema permite gestionar campañas sanitarias, jornadas y encuestas para mejorar la salud en Villa Argüello. Facilita decisiones basadas en datos georreferenciados y promueve el trabajo colaborativo con organizaciones sociales y el personal de salud.
